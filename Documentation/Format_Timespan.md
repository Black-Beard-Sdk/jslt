# Formatting Timespan

>'c': 00:00:00  
>'g': 0:00:00  
>'G': 0:00:00:00.0000000  
>'hh\:mm\:ss': 00:00:00  
>'%m min.': 0 min.  

>'c': -14.00:00:00  
>'g': -14:0:00:00  
>'G': -14:00:00:00.0000000  
>'hh\:mm\:ss': 00:00:00  
>'%m min.': 0 min.  

>'c': 01:02:03

>'G': 0:01:02:03.0000000  
>'hh\:mm\:ss': 01:02:03
