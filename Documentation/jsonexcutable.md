
# JSON executable

json.exe is a command line for execute easy

## Help
```CMD
    json.exe --help
```

## Loading csv
### Help
```CMD
    json.exe csv --help
```

## Loading excel
### Help
```CMD
    json.exe xls --help
```

### Help
```CMD
    json.exe schema --help
```

### Help
```CMD
    json.exe format --help
```

## Export
### Help
```CMD
    json.exe export --help
```

### Help
```CMD
    json.exe run --help
```
